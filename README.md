# 🏠 House Price Prediction using Linear Regression

## Overview

This project predicts house prices using the California Housing Dataset and the Linear Regression algorithm from Scikit-learn.

## Features

- Data Exploration (EDA)
- Data Visualization
- Linear Regression Model
- House Price Prediction
- Model Evaluation (MAE, MSE, RMSE, R²)
- Optional Tkinter GUI
- Project Report

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Joblib

## Dataset

California Housing Dataset from Scikit-learn.

## Project Structure

```text
task1_ml_linear_regression.ipynb
House_Price_Prediction_Report.pdf
house_price_model.pkl
house_price_ui.py
requirements.txt
README.md
```

## Results

The model predicts house prices using Linear Regression and is evaluated using:

- MAE
- MSE
- RMSE
- R² Score
